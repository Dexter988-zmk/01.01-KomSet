1 Заголовки

Заголовки являются одним из важнейши инструментов для структурирования текта.

1.1 Рекомендации по использованию

Вот некоторрые рекоменлдации по спользованию заголовков в HTML-странице.

1.1.1 Испольpование H1

Не рекомендуется, чтобы на одной страннице содержалось несколько заголовков верхнего уровня.

1.1.2 использование H2-H6

При использовании подзаголовков не рекомендуется пропускать уровни заголовков, то есть после заголовка H1 должен идти подзаголовок H2 и только потом подзаголовок H3.
